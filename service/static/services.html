manifest_service.rb

Web service implementation for outputting RDF for a Manifest file using N3 and JSON notation. This web service uses file system for retrieving the manifest information to be outputted.

Sample Links
  • Manifest_kaw-0032.n3 - Streams manifest content
  • Manifest_kaw-0032.json - Streams manifest content as json
  • Joseki Integration Test - Pulls content from Joseki, and translates it into JSON

manuscript_service.rb

Refactored manuscript service, as defined in DMSTechRESTfulWebServicesAPI

Sample Links
  • manuscript - Gets manuscript collection
  • manuscript.txt - Gets manuscript collection as plain text
  • 1.json - Gets manuscript #1 collection as json
  • 1.txt - Gets manuscript #1 collection as text
  • 1.n3 - Gets manuscript #1 collection as N3. Please note that this method might fail on Ruby 1.9.1 as there is a bug in RDF JSON library

prototype_service.rb

Prototype service for loading image FOXML data, applying an XSLT transformation to it to convert into RDF, then running a sample SPARQL query, and returning the result as JSON.

Sample Links
  • bd017cy0897 - Loads the content, applies the XSLT to it, then converts the XSLT output into an RDF graph and finally sparkles the result set.
